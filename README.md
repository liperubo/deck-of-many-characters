# Deck of Many Characters

Aplicação web para criar, organizar e evoluir fichas de personagens de RPG com foco em rapidez, narrativa e praticidade.

## Visão geral

O **Deck of Many Characters** permite montar um grimório de personagens com edição contínua, salvamento local automático e compartilhamento simples por JSON.

## Principais recursos

- Criação rápida de novas fichas.
- Edição completa de dados do personagem.
- Persistência local automática no navegador.
- Importação e exportação em JSON para backup e migração.
- Interface moderna com suporte a tema claro/escuro.

## Tecnologias

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Como executar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Scripts úteis

- `npm run dev` — inicia em modo desenvolvimento.
- `npm run build` — gera build de produção.
- `npm run start` — roda a build de produção.
- `npm run lint` — executa o lint do projeto.
